# TensorFlow Lab Experiments

This repository contains basic TensorFlow experiments performed using Google Colab.

## Experiments
- Creating Scalar, Vector, and Matrix Tensors  
- Creating a 3-D Tensor  
- Data Type, Shape, and Rank of Tensors  
- Tensor Operations (+, −, ×, ÷, Power)  
- Converting NumPy Arrays to TensorFlow Tensors  

## Tools Used
- Google Colab  
- Python  
- TensorFlow  
- NumPy
